# Weather Runner 🏃‍♂️🌦️

Бесконечный раннер с синхронизацией реальной погоды через API OpenWeatherMap.

## Особенности игры

### 🎮 Геймплей
- **Жанр**: Бесконечный раннер (Endless Runner)
- **Управление**: Пробел для прыжка (или тап на мобильных устройствах)
- **Цель**: Преодолеть максимальное расстояние, избегая препятствий

### 🌤️ Погодные условия
Игра адаптируется к реальной погоде в вашем городе:

| Погода | Визуал | Механика | Препятствия |
|--------|--------|----------|-------------|
| ☀️ Ясно | Яркое небо | Обычная скорость | Камни |
| 🌧 Дождь | Серое небо | **Скольжение** на лужах (ускорение x1.8) | Лужи |
| ❄️ Снег | Снегопад | **Замедление** (скорость 2.5), плавная гравитация | Сугробы |
| ⛈ Гроза | Тёмное небо | Высокая частота препятствий | Молнии (на земле и в воздухе) |

### 🎁 Бонусы
- ⭐ Звезда / ☂️ Зонт: +50 очков
- Появляются случайно вместо препятствий

## Установка и запуск

### Быстрый старт
1. Откройте файл `index.html` двойным кликом в браузере
2. Нажмите **"Узнать погоду и играть"**
3. Разрешите доступ к геолокации
4. Играйте!

### Настройка API ключа (опционально)

Для получения точных данных о погоде:

1. Зарегистрируйтесь на [OpenWeatherMap](https://openweathermap.org/api)
2. Получите бесплатный API ключ
3. Откройте `index.html` в текстовом редакторе
4. Найдите строку:
   ```javascript
   this.API_KEY = 'YOUR_API_KEY';
   ```
5. Замените `YOUR_API_KEY` на ваш ключ

**Без API ключа** игра будет использовать случайную погоду (fallback режим).

## Архитектура проекта

### Классы (ES6)

```
WeatherService - Работа с Weather API
├── getCurrentWeather() - получение данных
├── mapWeatherToGameState() - маппинг погоды
└── getFallbackWeather() - резервный режим

Player - Игрок
├── jump() - прыжок
├── update() - физика (гравитация)
└── draw() - рендеринг

Obstacle - Препятствия
├── rock, puddle, snowbank, lightning
└── collision detection

Collectible - Бонусы
├── star, umbrella
└── +50 очков при сборе

SnowParticle - Частицы снега
└── анимация снегопада

Game - Основной игровой цикл
├── gameLoop() - requestAnimationFrame
├── update() - логика
├── draw() - рендеринг
└── checkCollisions() - AABB коллизии
```

### Технические детали

- **Canvas**: 800x400px
- **Game Loop**: `requestAnimationFrame` (~60 FPS)
- **Физика**: Гравитация 0.5 (снег: 0.3)
- **Коллизии**: AABB (Axis-Aligned Bounding Box)
- **Хранение рекорда**: `localStorage`

## Troubleshooting

### Игра не запрашивает геолокацию
- Проверьте настройки браузера (разрешения)
- Убедитесь, что используете HTTPS или localhost

### Погода всегда одинаковая
- Проверьте API ключ в коде
- Убедитесь, что ключ активирован (может занять несколько минут после создания)

### Низкая производительность
- Закройте другие вкладки
- Обновите браузер до последней версии
