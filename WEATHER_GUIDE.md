# 🌍 Руководство по использованию реальной погоды

## Быстрый старт (1 минута)

### Шаг 1: Откройте игру
Дважды кликните на `index.html`

### Шаг 2: Выберите способ получения погоды

#### Вариант A: Поиск по городу ⭐ Рекомендуемый
1. Введите название города: `Москва`, `London`, `New York`
2. Нажмите **Enter** или кнопку **"🔍 Найти город"**
3. Подождите 1-2 секунды
4. Увидите погоду и название города
5. Игра начнётся автоматически

#### Вариант B: Моё местоположение
1. Нажмите **"📍 Моё местоположение"**
2. Разрешите доступ к геолокации
3. Город определится автоматически
4. Игра начнётся автоматически

#### Вариант C: Случайная погода
1. Нажмите **"Играть со случайной погодой"**
2. Мгновенный старт без ожидания

## Подсказки

### 💡 Для быстрого поиска:
- Используйте **Enter** после ввода города
- Начните вводить название - браузер может предложить автодополнение
- Используйте английские названия для международных городов

### 💡 Если город не находится:
- Проверьте правильное написание
- Попробуйте английское название
- Добавьте страну через запятую: `London, UK`
- Проверьте интернет соединение

### 💡 Для лучшей точности:
- Используйте крупные города (они точнее в базе данных)
- Избегайте маленьких посёлков
- Для России используйте русские названия

## Troubleshooting

### ❌ "Город не найден"
**Решение:**
1. Проверьте написание
2. Попробуйте английское название
3. Добавьте страну: `City, Country`
4. Проверьте интернет соединение

### ❌ "Не удалось определить местоположение"
**Решение:**
1. Разрешите геолокацию в браузере
2. Проверьте настройки приватности
3. Используйте HTTPS (не file://)
4. Попробуйте поиск по городу вместо геолокации

### ❌ Погода не загружается
**Решение:**
1. Проверьте интернет соединение
2. Подождите несколько секунд
3. Обновите страницу
4. Попробуйте другой браузер

## API Limits

**Бесплатный тариф OpenWeatherMap:**
- ✅ 60 запросов в минуту
- ✅ 1,000,000 запросов в месяц
- ✅ Более чем достаточно для личного использования

**Текущее использование:**
- ~1 запрос при каждом запуске игры
- Можно играть тысячи раз без превышения лимита
