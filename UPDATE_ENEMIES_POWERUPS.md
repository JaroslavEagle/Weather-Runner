# Обновление: Враги, Power-Ups и Улучшенный Геймдизайн 🎮⚔️

## Что нового (2026-04-19)

### ✅ Система врагов для каждой погоды

Теперь в игре появились **уникальные враги** для каждого погодного условия:

#### ❄️ Yeti (Снежный человек) - SNOW
- **Внешность**: Белое мохнатое существо с красными глазами
- **Поведение**: Быстро бежит навстречу игроку
- **Скорость**: 3-5 px/frame быстрее препятствий
- **Анимация**: Руки качаются при беге
- **Опасность**: Мгновенная смерть при столкновении

#### ☀️ Drowned (Утопленник) - SUNNY  
- **Внешность**: Полупрозрачное водное существо с зелёными светящимися глазами
- **Поведение**: Появляется парадоксально на солнце
- **Скорость**: 3-5 px/frame
- **Особенности**: Водоросли/капли над головой
- **Лор**: Дух воды, появляющийся в жару

#### 🌧 Wet Traveler (Промокший путник) - RAIN
- **Внешность**: Фигура в тёмном плаще с капюшоном
- **Поведение**: Бредёт под сломанным зонтом
- **Скорость**: 3-5 px/frame
- **Детали**: Грустное лицо, мокрый плащ
- **Атмосфера**: Меланхоличный враг дождливой погоды

#### ⛈ Random Enemy - STORM
- **Механика**: В грозу может появиться любой из трёх врагов
- **Вероятность**: Равномерное распределение (33% каждый)
- **Непредсказуемость**: Добавляет сложность штормовой погоде

### ✅ Система Power-Ups (Усилений)

Четыре типа усилений для улучшения геймплея:

#### 🛡️ Shield (Щит)
- **Эффект**: Защищает от одного удара (враг или препятствие)
- **Визуал**: Синий круг вокруг игрока
- **Длительность**: Постоянный до использования
- **Индикатор**: Полупрозрачный синий ореол вокруг персонажа
- **Стратегия**: Сохранять для сложных моментов

#### 🧲 Magnet (Магнит)
- **Эффект**: Притягивает бонусы в радиусе 200px
- **Длительность**: 10 секунд (600 кадров)
- **Визуал**: Красный ромб с символом магнита
- **Механика**: Бонусы летят к игроку автоматически
- **Стратегия**: Использовать в зонах с密集ными бонусами

#### ⭐ Double Score (Двойные очки)
- **Эффект**: Все бонусы дают x2 очков (100 вместо 50)
- **Длительность**: 10 секунд (600 кадров)
- **Визуал**: Золотой ромб с надписью "x2"
- **Стратегия**: Активировать когда много бонусов на экране

#### ⏱️ Slow Motion (Замедление времени)
- **Эффект**: Скорость игры уменьшается на 50%
- **Длительность**: 8 секунд (480 кадров)
- **Визуал**: Фиолетовый круг с часами
- **Механика**: Все объекты движутся медленнее
- **Стратегия**: Использовать в сложных ситуациях

### ✅ Возврат в главное меню

**Новая кнопка "🏠 В меню"** на экране Game Over:

**Функционал:**
- Полный сброс игры

- Сброс счётчиков и таймеров
- Возврат на стартовый экран

### Spawn Logic

```javascript
spawnObstacles(now) {
    // Препятствия: каждые 1-3 секунды
    if (now - this.lastObstacleTime > obstacleInterval) {
        const rand = Math.random();
        if (rand < 0.15) {
            this.spawnCollectible();    // 15% шанс
        } else if (rand < 0.20) {
            this.spawnPowerUp();        // 5% шанс
        } else {
            this.spawnObstacle();       // 80% шанс
        }
    }
    
    // Враги: каждые 8-12 секунд
    if (now - this.lastEnemyTime > 8000-12000ms) {
        this.spawnEnemy();
    }
}
```

### Collision Handling

```javascript
checkCollisions() {
    // Столкновение с врагом/препятствием
    if (collision) {
        if (this.hasShield) {
            this.hasShield = false;     // Щит поглощает удар
            this.showNotification('🛡️ Щит использован!');
            removeObstacle();
        } else {
            this.gameOver();            // Без щита - смерть
        }
    }
    
    // Столкновение с power-up
    if (powerUpCollision) {
        this.activatePowerUp(powerUp.type);
        removePowerUp();
    }
}
```

## Геймдизайн принципы

### 1. Risk vs Reward (Риск против Награды)
- **Power-ups** появляются редко (5% шанс)
- **Враги** появляются предсказуемо (8-12 сек)
- **Бонусы** часто, но требуют риска (прыжки)

### 2. Progressive Difficulty (Прогрессивная сложность)
- Враги движутся **быстрее** препятствий
- Чем дольше играешь, тем больше врагов
- Power-ups помогают выжить дольше

### 3. Weather-Themed Enemies (Тематические враги)
- Каждый враг соответствует своей погоде
- Визуальный дизайн отражает атмосферу
- Лор: парадоксальные враги (утопленник на солнце)

### 4. Strategic Depth (Стратегическая глубина)
- **Щит**: Когда использовать? Сохранять или сразу?
- **Магнит**: Позиционирование важно
- **Двойные очки**: Тайминг активации
- **Замедление**: Экстренная ситуация

### 5. Visual Feedback (Визуальная обратная связь)
- Индикатор щита вокруг игрока
- Уведомления при активации power-up
- Таймеры с уведомлениями об окончании
- Анимации покачивания для power-ups

## Player Psychology

### Variable Rewards (Переменные награды)
- Power-ups появляются случайно (5% шанс)
- Создаёт азарт и ожидание
- Dopamine hit при получении редкого усиления

### Achievement Motivation (Мотивация достижения)
- High score tracking
- Survival time challenge
- Perfect run attempts

### Loss Aversion (Избегание потерь)
- Shield создаёт чувство безопасности
- Игроки берегут щит "на потом"
- Эмоциональная привязанность к прогрессу

### Flow State (Состояние потока)
- Progressive difficulty
- Clear goals (survive, collect)
- Immediate feedback (notifications)
