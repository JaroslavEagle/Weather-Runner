# 🎮 Weather Runner - At a Glance

Краткий обзор проекта для быстрого ознакомления (2 минуты чтения).

---

## Что это?

**Weather Runner** — бесконечный раннер, который синхронизируется с реальной погодой через API.

🌤️ Дождь на улице → лужи и скольжение в игре  
❄️ Снег на улице → замедление и снегопад  
⛈ Гроза на улице → молнии в воздухе  

---

## Быстрый старт

1. Откройте `index.html` в браузере
2. Нажмите "Узнать погоду и играть"
3. Разрешите геолокацию
4. Играйте! (Пробел = прыжок)

---

## Ключевые особенности

✅ Реальная погода влияет на геймплей  
✅ 4 погодных состояния с уникальными механиками  
✅ Кроссплатформенность (Desktop + Mobile)  
✅ Система очков и рекордов  
✅ Полностью бесплатно и open source  

---

## Технологии

- **Frontend:** HTML5 Canvas, CSS3, JavaScript ES6+
- **API:** OpenWeatherMap (геолокация + погода)
- **Storage:** localStorage (рекорды)
- **Dependencies:** None (vanilla JS)

## Основные механики

### Погодные состояния

| Погода | Сложность | Уникальная механика |
|--------|-----------|---------------------|
| ☀️ Ясно | ⭐ Легко | Стандартный режим |
| 🌧 Дождь | ⭐⭐ Средне | Скольжение на лужах |
| ❄️ Снег | ⭐⭐ Средне | Замедление, плавное падение |
| ⛈ Гроза | ⭐⭐⭐ Сложно | Молнии в воздухе |

### Управление
- **Desktop:** Пробел = прыжок
- **Mobile:** Тап = прыжок

### Цель
Набрать максимальное количество очков, избегая препятствий.

## Архитектура (упрощённо)

```
Game (main controller)
├── Player (физика, рендер)
├── Obstacle (препятствия)
├── Collectible (бонусы)
├── WeatherService (API)
└── SnowParticle (частицы)
```

**Pattern:** Game Loop с requestAnimationFrame (~60 FPS)
